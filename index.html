Project Description

We were first introduced to Angular in SE 396 (assignment six). In this class we used Angular to create a corporation finder. This included a search feature and the ability to drill down into the corportation details. While it was nice to start working with a new language, this assignment only provided us with a taste of what Angular could do. For this class I would like to study Angular in more depth. My goal will be to create a website for an anual memorial run. I plan to use the MEAN stack to create this application.

Application Details:

  • Provide information about the cause
  • Display pictures of past runs and events
  • Provide a contact form for users
  • Provide a sponsor form for potential sponsors
  • Interactive course information for the upcoming run
  • Handle registration for upcoming run and events
